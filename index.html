Our grid works with almost any device and browser.
We designed for quick and easyly using and easy to learn.
You can create complex layouts very easy, and responsive.

We divide the screen in 10 parts.
and there are four screen size, Small, medium, large, elarge
Exmple: if you grid in large, elarge screen still remain the same, but when open in the small and medium screen, the width will go to 100% and drop the another element down.
But if you grid in small, everything is still the same no matter what the screen is. Look at examples below:

small-7

Syntax
